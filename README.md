# auxHelper
Webapp that allows people to request and vote on songs for a group queue linked to and playing on a spotify account.


Dependencies:
- nodejs
- npm

## Getting Started

Installs
- Clone repo
- Make sure you have node.js and npm
- In bash or cmd navigate to the root folder 
- run ```npm install``` to install all dependencies


How to run:
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

you can then view the project at http:/localhost:3000
