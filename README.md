# auxHelper
Webapp that allows people to request and vote on songs for a group queue linked to and playing on a spotify account.


Dependencies:
- nodejs
- npm

## Getting Started

How to get the dependencies required:
- Clone the repo
- Navigate to the root folder in bash
- run ```bash npm install``` or whatever based off your package manager.

How to run:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

you can then view the project at http:/localhost:3000
